Test de persistance Zustand

Ouvrez cette page dans votre navigateur pour tester:

  1. Allez sur http://localhost:3000
  2. Changez le mode dark/light avec le bouton
  3. Allez sur /setup et configurez quelques paramètres
  4. Connectez des réseaux sociaux (fake)
  5. Rechargez la page (F5)
  6. Vérifiez que le thème et les connexions sont conservés

Vérification dans la console

Dans les DevTools du navigateur (F12), onglet Application > Local Storage:

  • preferences-storage: contient darkMode, sidebarOpen, language
  • project-storage: contient brandIdentity, goals, setupCompleted
  • integrations-storage: contient les connexions réseaux sociaux
